


#  Join Member System - Nawah Project

هذا المشروع هو نظام تعريفي وتسجيل للأعضاء كجزء من **مبادرة نواة (Nawah Project)**. تم التركيز فيه على تقديم واجهة مستخدم (UI) عصرية وسريعة الاستجابة باستخدام تقنيات الـ Front-End الحديثة.

##  التقنيات المستخدمة (Tech Stack)
* **HTML5**: لبناء الهيكل الإنشائي للصفحات.
* **Tailwind CSS**: للتنسيق المتقدم والتصميم المتجاوب (Responsive Design).
* **JavaScript**: لإضافة التفاعلية والمنطق البرمجي للواجهة.
* **Git & GitHub**: لإدارة النسخ والعمل الجماعي.

##  هيكل المشروع (Project Structure)
المشروع منظم بطريقة تسهل الصيانة والتطوير:
* `src/pages/`: تحتوي على الصفحات الأساسية (الرئيسية، الأخبار، التخصصات).
* `src/css/`: تضم ملفات التنسيق والصور والفيديوهات.
* `src/js/`: تحتوي على ملفات البرمجة التفاعلية.

## المميزات (Features)
* **Custom Design**: استخدام ألوان وهوية بصرية مخصصة عبر `tailwind.config.js`.
* **Responsive Layout**: الموقع يعمل بكفاءة على جميع الشاشات (Mobile, Tablet, Desktop).
* **Nawah Space Section**: سكشن خاص يعرض مساحة المشروع بأسلوب بصري جذاب.
* **Specializations Page**: عرض تصنيفات التخصصات بطريقة منظمة.

## طريقة التشغيل (How to Run)
1. قم بتحميل المشروع (Clone) من الرابط.
2. افتح مجلد `src/pages`.
3. قم بتشغيل ملف `home.html` في أي متصفح.




#  Join Member System - Nawah Project

This project is an introductory and registration system for members, created as part of the **Nawah Project**. The focus is on providing a modern and responsive User Interface (UI) using the latest Front-End technologies.

##  Tech Stack
* **HTML5**: To build the structure of the web pages.
* **Tailwind CSS**: For advanced styling and Responsive Design.
* **JavaScript**: To add interactivity and logic to the interface.
* **Git & GitHub**: For version control and collaboration.

##  Project Structure
The project is organized to make maintenance and development easy:
* `src/pages/`: Contains the main pages (Home, News, Specializations).
* `src/css/`: Includes stylesheets, images, and videos.
* `src/js/`: Contains interactive programming files.

##  Features
* **Custom Design**: Custom colors and visual identity via `tailwind.config.js`.
* **Responsive Layout**: The website works perfectly on all devices (Mobile, Tablet, Desktop).
* **Nawah Space Section**: A special section displaying the project space with an attractive visual style.
* **Specializations Page**: Organized display of different specialization categories.

##  How to Run
1.  **Clone** the project from the repository link.
2.  Open the `src/pages` folder.
3.  Run the `home.html` file in any web browser.
